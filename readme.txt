== Description ==

Portfolio (and their categories) in the Pexeto Themes will be synchronized with Lightroom (your photos, collections, and collection sets). This magic requires also the WP/LR Sync plugin to be installed.

Languages: English.

== Installation ==

1. Install WP/LR Sync (http://apps.meow.fr/wplr-sync)
2. Install a Pexeto theme
3. Install Pexeto for Lightroom (this plugin)
4. Synchronize everything :)
